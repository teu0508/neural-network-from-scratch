# neural-network-from-scratch

This project is a simple implementation of a neural network built from scratch in Python.

The goal is to understand how neural networks work by implementing the core logic manually, without using high-level machine learning libraries.

The model will be trained on the MNIST dataset, with the objective of achieving a good accuracy on both the MNIST test set and the FashionMNIST dataset.
